LaterNest

一个本地优先的 Chrome 稍后阅读工作台。把当前网页保存成待处理内容，再用时间线、周视图和批量整理把它们带回眼前。

Install from source Read the story

为什么做它

收藏不是终点。真正困难的是在合适的时候重新看见、整理并完成。LaterNest 让稍后阅读更像一个安静的收件箱，而不是另一个被遗忘的列表。

核心功能

快速保存

从当前标签页保存标题、URL、来源域名、备注、分类和待看时间。

整理工作台

在 Dashboard 中搜索、筛选、完成、延后、删除和批量整理保存内容。

时间线与周视图

按日期回看，也可以把一周积累的内容集中梳理出来。

本地优先，不默认上传

LaterNest 没有账号系统，也没有远程数据库。你的保存内容默认留在自己的 Chrome 本地存储里。只有当你主动配置 Feishu/Lark Webhook 时，才会把未看清单发送到你自己的机器人。

安装

下载仓库后打开 chrome://extensions/，开启开发者模式，选择“加载已解压的扩展程序”，然后选中 LaterNest 文件夹。
