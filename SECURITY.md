# Security Notes

LaterNest is intended to be installed locally from source.

## Sensitive Data

Do not commit:

- Real Feishu/Lark webhook URLs
- Exported `chrome.storage` data
- Personal browsing records
- Internal company links
- Screenshots containing private content

## Reporting Issues

If you publish this repository publicly, use GitHub Issues for non-sensitive bugs.

For private security concerns, configure a private contact method before encouraging public reports.
