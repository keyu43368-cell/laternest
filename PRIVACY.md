# Privacy Policy

LaterNest is built as a local-first Chrome extension.

## Data Stored Locally

LaterNest stores the following data in your browser through `chrome.storage.local`:

- Saved page title
- Saved page URL
- Page domain
- Optional note
- Category
- Status
- Reminder time
- Extension settings
- Optional Feishu/Lark webhook URL

This data is stored on your device inside Chrome. LaterNest does not provide a hosted backend service.

## Data Sent Externally

LaterNest only sends saved link data externally when you explicitly configure Feishu/Lark sync in the settings page.

If configured, LaterNest sends selected pending links to the webhook URL that you provide. The webhook belongs to you, and delivery is made directly from the extension to Feishu/Lark.

## Data Not Collected

LaterNest does not collect:

- Accounts
- Passwords
- Payment information
- Full browsing history
- Analytics events
- Advertising identifiers
- Personal data on a remote LaterNest server

## Webhook Safety

A Feishu/Lark webhook URL is sensitive. Do not commit it to GitHub, share it publicly, or paste it into screenshots.

## Contact

If you fork or redistribute LaterNest, update this privacy policy with your own contact and support details.
